System Properties

Static / Dynamic

  • Also known as memoryless / memory systems respectively
  • If the output of the system at any time instance depends only on the present values of the input {x(t)}
  • y(t) = x(t) : static
  • y(t) = x(t-c) : dynamic
  • y(t) = x(t+c) : dynamic
  • y(t) = [differential of x] : dynamic

Causal / Non-Causal

  • If the output of the system at any time instance depends only on the present or past values of the input {x(t)}
  • Response of the system does not begin before the input function is applied
  • The impulse response of the system for a causal system is zero for t < 0
  • if y(t) = [integral of x(t)] then y(t) is non causal
