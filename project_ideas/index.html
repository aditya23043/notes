Keyboard

An ergonomic ortholinear mono-board keyboard with wired and wireless functionalities alongside some neat features, some of which are listed below

  • Dictionary
    checks definition and other info about the word being typed without actually sending keypresses using https://api.dictionaryapi.dev/api/v2/entries/en/{word}
  • Quotes
    Displays inspirational quotes on the screen which is shown by default in normal mode using https://zenquotes.io/api/random
  • Time
    by default on normal mode which is updated every minute since we do not need precise time; using http://worldtimeapi.org/api/timezone/Asia/Kolkata
  • Vim Mode
    When enabled, the keyboard will keep track of the current mode (normal/visual/insert/replace/select/command) and will send keypresses in accordance with the corresponding vim counterpart

Physical Password Manager Device using MCU

A physical usb device which plugs in your device and enters your passwords for you so that you dont have to look and then type

  • Locally storing passwords on the device
  • No interfacing from the system
  • Acts as a simple HID when looking from the system
  • All interfacing / configuration from user part to be done through onboard buttons and screen on the device
  • Size should be ideally like a thumb drive
  • Master key?
