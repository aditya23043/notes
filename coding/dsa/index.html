Data Structures and Algorithm

Data Structure

A data structure is a way of organizing data so that it can be used effectively.

Abstract Data Structures

An abstraction of a data structure which provides only the interface to which a data structure must adhere to. The interface does not give any specific details about how something should be implemented or in which programming language.

Examples

Abstraction Implementation
List Dynamic Array / Linked List
Queue Linked list / Stack / Array
Map Tree map / Hash map / Hash table

Big-O Notation

Gives an upper bound of the complexity in the worst case, helping to quantify performance as the input size becomes arbitrarily large

Type Complexity
Constant O(1)
Logarithmic O(logn)
Linear O(n)
Linearithmic O(nlogn)
Quadric O(n^2)
Cubic O(n^3)
Exponential O(b^n)
Factorial O(n!)

Big-O Properties

  • O(n + c) = O(n)
  • O(c * n) = O(n) : c > 0

Big-O examples

  • Finding all subsets of a set : O(2^n)
  • Finding all permutations of a string : O(n!)
  • Soring using merge sort : O(nlogn)
  • Iterating over all cells of an n x m matrix : O(n*m)

Static Arrays
