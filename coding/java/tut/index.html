Data Types

Data Type Size Description
int 4 bytes -2 bil to +2 bil
long 8 bytes -9 quintillion to +9 quintillion
short 2 bytes -32,768 to +32,767
byte 1 bytes -128 to 127
float 4 bytes fractional number upto 5-6 digits and requires an f at the end
double 8 bytes double precision floating point number upto 15 digits
bool 1 byte does not return 0 or 1. Its simply true or false
char 2 byte requires single quotes
String varies requires double quotes
  • Note: String is a reference data type whereas every other data type is primitive datatype
			
	public class Main {
		public static void main(String[] args) {
			
			int num1 = 4;
			long num2 = 8;
			short num3 = 2;
			byte num4 = 1;

			float cgpa = 7.3f;
			double us_gpa = 3.0;

			boolean wants_to_get_placed_on_campus = true;

			char grade = 'B';
			String name = "Aditya Gautam";

		}
	}

Literals

  • You can pass in hexadecimal, binary and exponential numbers to int data type as well
			
	public class Main {
		public static void main(String[] args) {
			int num1 = 0x7E;
			int num2 = 0b0101;
			int num3 = 12e10;
			int num4 = 10_00_00_00_000;
		}
	}

String

    
  String something = new String("something");
  String foo = "foo";
  • Both statements are the same thing

String Buffer

    
  StringBuffer something = new StringBuffer("something");
  • StringBuffer is a peer class of String, it is mutable in nature and it is thread safe class , we can use it when we have multi threaded environment and shared object of string buffer i.e, used by mutiple thread. As it is thread safe so there is extra overhead, so it is mainly used for multithreaded program.
