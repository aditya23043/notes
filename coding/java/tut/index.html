Data Types

Data Type Description
int 4 bytes limit
long 8 bytes limit
short 2 bytes limit
byte 1 bytes limit
float requires an f at the end and 4 bytes limit
double double precision floating point number and 8 bytes limit
bool does not return 0 or 1. Its simply true or false
char requires single quotes
String requires double quotes 
			
				public class Main {
					public static void main(String[] args) {
						
						int num1 = 4;
						long num2 = 8;
						short num3 = 2;
						byte num4 = 1;

						float cgpa = 7.3f;
						double us_gpa = 3.0;

						boolean wants_to_get_placed_on_campus = true;

						char grade = 'B';
						String name = "Aditya Gautam";

					}
				}

Literals

  • You can pass in hexadecimal, binary and exponential numbers to int data type as well
			
				public class Main {
					public static void main(String[] args) {
						int num1 = 0x7E;
						int num2 = 0b0101;
						int num3 = 12e10;
						int num4 = 10_00_00_00_000;
					}
				}
