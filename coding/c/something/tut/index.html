Int Division VS Float Division

Integer

#include <stdio.h> int main(){ int x = 12; int y = 5; int y = x / y; return 0; }

Float

#include <stdio.h> int main(){ int x = 12; int y = 5; int y = x / (float)y; return 0; }

Make any one or both of the operators float explicitly
