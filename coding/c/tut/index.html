Constant Value

  • constant values cannot be changed after declaration
  • naming convention : all letters => uppercase
#include <stdio.h> int main() { const float PI = 3.14159; // PI = 5; // Reassigning a value to a const will result in an error printf("%d\n", PI); return 0; }

Int Division VS Float Division

Integer

#include <stdio.h> int main(){ int x = 12; int y = 5; int y = x / y; }

Float

#include <stdio.h> int main(){ int x = 12; int y = 5; int y = x / (float)y; }
